КБЖУ-трекер
(Калории, Белки, Жиры, Углеводы)

Автор: Кузнецов Илья Дмитриевич

Описание: Бот спрашивает базовые данные (пол, возраст, рост, вес) и цель (похудение, набор, поддержание). На основе этого рассчитывает дневную норму калорий и БЖУ и ежедневно помогает её соблюдать: предлагает целевые значения на день, принимает каждый приём пищи (например, “сосиски 3 шт”), автоматически суммирует КБЖУ за день и показывает остаток до цели. Если пользователь вышел за дневной лимит, бот рассчитывает, сколько ориентировочно калорий нужно сжечь (например, ходьба/бег) для компенсации. Дополнительно бот учитывает повседневную трату калорий (бытовая активность, шаги/ходьба) для более точной оценки фактического дефицита или профицита за день.

Наши преимущества:

Нулевой барьер входа: Бот запускается одной кнопкой в мессенджере, который уже открыт у пользователя 24/7.
Экономия времени: Взаимодействие происходит через чат — самый естественный способ общения.
Всегда под рукой: Трекер калорий работает эффективно только тогда, когда запись вносится «здесь и сейчас».
Кроссплатформенность: Бот одинаково идеально работает на смартфоне, планшете и компьютере
