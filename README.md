Автор: Кузнецов Илья 
Тема: КБЖУ-трекер
Описание: Бот спрашивает базовые данные (пол, возраст, рост, вес, уровень активности) и цель (похудение, набор, поддержание). На основе этого рассчитывает дневную норму калорий и БЖУ и ежедневно помогает её соблюдать: предлагает целевые значения на день, принимает каждый приём пищи в свободной форме (например, “сосиски 3 шт”), автоматически суммирует КБЖУ за день и показывает остаток до цели. Если пользователь вышел за дневной лимит, бот рассчитывает, сколько ориентировочно калорий нужно сжечь (например, ходьба/бег) для компенсации. Дополнительно бот учитывает повседневную трату калорий (бытовая активность, шаги/ходьба) для более точной оценки фактического дефицита или профицита за день.
